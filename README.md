# Sales Performance Analysis Dashboard – Excel Mini Project

## Project Overview

This project is an interactive **Sales Performance Analysis Dashboard** created using **Microsoft Excel**. The dashboard helps analyze sales data across different regions and product categories using charts, pivot tables, and slicers.

The objective of this project is to track:

* Quantity Sold
* Profit Performance
* Discount Analysis
* Regional Sales Distribution
* Product-wise Sales Insights

This dashboard provides a clear visual representation of business performance and supports better decision-making.

## Tools & Features Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Data Cleaning
* Dashboard Design

## Dashboard Insights

### 1. Quantity Sold by Region

Displays the total quantity sold across:

* East
* North
* South
* West

### 2. Profit Analysis

Shows profit comparison between regions to identify high-performing areas.

### 3. Quantity Sold by Product Category

Analyzes sales contribution from categories such as:

* Clothing
* Electronics
* Furniture
* Groceries
* Sports

### 4. Discount Analysis

Tracks total discounts provided across different product categories.

### 5. Product & Region Sales Comparison

Compares quantity sold and profit across regions and products.

## Key Objectives

* Monitor overall sales performance
* Identify top-performing regions
* Analyze category-wise sales trends
* Understand discount impact on profit
* Create an interactive business dashboard
